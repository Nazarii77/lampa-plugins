Interface MOD для Lampa

Версия 2.1.0

Описание плагина

Interface MOD - это плагин для приложения Lampa, который улучшает интерфейс и добавляет множество новых функций для более комфортного использования. Плагин разработан с вниманием к деталям и полностью настраивается через меню настроек.

Основные функции

  • Информация о сезонах и сериях в двух строках на постере
  • Цветные рейтинги фильмов и сериалов
  • 5 красивых тем оформления интерфейса
  • Отображение всех кнопок в карточке фильма/сериала
  • Изменение лейблов типа контента на "Фильм" и "Сериал"

Установка плагина

Для установки плагина в Lampac добавьте в lampac/plugins/lampainit.my.js следующий код:

var plugins = Lampa.Plugins.get(); var plugins_add = [ {initiale}, //..............Ваш код если есть............... {"url": "https://bywolf88.github.io/lampa-plugins/interface_mod.js","status": 1,"name": "Interface MOD","author": "bywolf"}, ];

Для установки плагина в Lampa

Зайти в настройки — расширения — добавить плагин и ввести адрес плагина:

https://bywolf88.github.io/lampa-plugins/interface_mod.js
