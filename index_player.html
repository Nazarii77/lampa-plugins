Player MOD
Версия 1.0.0
Плагин для улучшения плеера в приложении Lampa. Добавляет новые функции воспроизведения, улучшает качество видео и звука, а также предоставляет дополнительные настройки для комфортного просмотра.
  • ✦ Улучшенное качество воспроизведения видео
  • ✦ Расширенные настройки звука
  • ✦ Поддержка HDR и 4K
  • ✦ Автоматическая настройка качества
  • ✦ Улучшенная работа с субтитрами
  • ✦ Гибкие настройки через меню

Установка плагина

Для установки плагина в Lampac добавьте в lampac/plugins/lampainit.my.js следующий код:

var plugins = Lampa.Plugins.get();
var plugins_add = [
  {initiale},
  //..............Ваш код если есть...............
  {"url": "https://bywolf88.github.io/lampa-plugins/player_mod.js","status": 1,"name": "Player MOD","author": "bywolf","version": "1.0.0"},
];

Для установки плагина в Lampa

Зайти в настройки — расширения — добавить плагин и ввести адрес плагина:

https://bywolf88.github.io/lampa-plugins/player_mod.js
